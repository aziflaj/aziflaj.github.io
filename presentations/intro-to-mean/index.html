What is MEAN?

A full-stack development method using only JavaScript

It uses MongoDB as database

It uses Express as a Node.js framework

It uses AngularJS as a front-end framework

What is Node.js?

Lightweight, event-driven, back-end Javascript Framework

Built on Chrome's V8

Fast!

How Fast?

According to PayPal:

  • Double the req/sec vs. the Java app
  • 35% decrease in the average response time

What is Express.js?

  • Fast, unopinionated, minimalist web framework for node.
  • Focused on high performance
  • HTTP helpers

An Express Example


var express = require('express'),
    app = express();

app.get('/', function (req, res) {
  res.send('Hello World');
});

app.listen(3000);

Starting a MEAN app

Start with a package.json file


{
  "name": "MyMeanApp",
  "version": "1.0.0",
  "description": "This is an awesome app built with MEAN",
  "main": "server.js",
  "scripts": {
    "start": "node ./server"
  },
  "author": "John Doe",
  "license": "MIT"
}

Install dependencies


$ npm install express --save
$ npm install monk --save

Start the webapp


var express = require('express'),
    app = express();

app.get('/', function(req, res) {
    res.send("Hello! You are ready to develop using Express");
});

app.listen(3000, function() {
	console.log("Server ready. Listening on port 3000");
});
